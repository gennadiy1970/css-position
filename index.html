Position

Позволяет выбрать альтернативные правила позиционирования элементов: разместить элементы вне потока выполнения и дать широкие возможности для создания эффектов (например эффект выпадающего меню) и анимации

значения свойства position:

  • static значение по умолчанию для всех элементов
  • relative
  • absolute
  • fixed
  • sticky
  • z-index

Основные примеры использования

  • Модальное окно
  • Выпадающее меню
  • Текст над изображением
